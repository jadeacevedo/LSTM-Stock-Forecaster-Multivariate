# LSTM-Stock-Forecaster-Multivariate 
(current project i am still working on, currently unfinished)
A deep learning framework for stock price forecasting using LSTM and RNN architectures. This project implements a multivariate time-series approach, integrating technical indicators (RSI, EMA) and min-max scaling to capture complex market dependencies and mitigate the vanishing gradient problem in financial data
